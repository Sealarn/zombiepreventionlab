Zombie Outbreak Prevention
SAVE THE WORLD IN MIXED-REALITY

Project

What

Zombie Outbreak Prevention is an interactive experience in MR where teenagers can solve puzzles.

How

Every user is handed a set of sick-ass goggles where hints to solve different puzzles is provided through an application in MR. The different hint models is rendered with customized markers.

Why

The goal of this project was to inspire and encourage young teenage girls into studying STEM fields.
